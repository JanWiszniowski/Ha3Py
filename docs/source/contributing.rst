############
Contributing
############

Ha3Py estimation development happens on
`GitHub <https://github.com/JanWiszniowski/ha3py>`_.

I'm very open to contributions: if you have new ideas, please open an
`Issue <https://github.com/JanWiszniowski/ha3py/issues>`_.
Don't hesitate sending me pull requests with new features and/or bugfixes!
